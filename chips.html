Чипы

Чипы используются для визуализации выбираемых вариантов.

Когда использовать?

Чипы или чекбоксы/тогглы?

Чипы визуально нагляднее чекбоксов и тогглов, поскольку они заметнее. Также чипы занимают меньше вертикального пространства, поскольку являются строковым элементом. Поэтому в интерфейсе предпочтение отдается именно чипам, а тогглы и чекбоксы рекомендуется использовать, например, в формах для заполнения.

Чипы или бейджи?

Чипы — это элемент ввода. Их добавляет, выбирает, редактирует и удаляет пользователь.

Бейдж — это элемент визуализации. Их видит пользователь, но не может отредактировать.

Иногда чип — это бейдж в режиме редактирования. Например:

  1. Пользователь создаёт новую игру и прописывает с помощью чипов её характеристики: командная, синхронная, с картинками, со звуком. Это чипы.
  2. После публикации игры пользователь видит её карточку с характеристиками. Это бейджи.
  3. Он нажимает на «Редактировать игру». Это снова чипы.

Анатомия

Спецификация

Размер и контейнер

Рекомендуется использовать размер Medium и отображать чипы строкой. Размер Small используется только в случае, когда необходимо отобразить чипы внутри Строки ввода.

Содержание

Рекомендуется создавать теги не длиннее 19 символов. Лучше использовать слова или словосочетания и не использовать предложения, знаки препинания и названия компаний.

Вёрстка

Чипы — это строковый элемент, поэтому они располагаются друг за другом и переносятся на следующую строку как текст. Они выравниваются по левому краю.

Вертикальный и горизонтальный промежуток:

  • 8px для Medium
  • 4px для Small

Как выбрать вариант?

Удаляемые

Для визуализации многочисленных и уникальных опций, выбранных пользователем.

<body>
    <pre><code class="language-html">...</code></pre>
</body>
.punctuation {    color: black;}
.bracket {    color: blue;}
.css .property { color:22px;}
.css .numerical { color:seagreen;}
.css .letter { color:cornflowerblue;}

Неудаляемые

Для визуализации малочисленных и заранее определенных опций, выбранных пользователем.

<body>
    <pre><code class="language-html">...</code></pre>
</body>
.punctuation {    color: black;}
.bracket {    color: blue;}
.css .property { color:22px;}
.css .numerical { color:seagreen;}
.css .letter { color:cornflowerblue;}

С выпадающим меню

Для визуализации и выбора опций внутри таблицы.

<body>
    <pre><code class="language-html">...</code></pre>
</body>
.punctuation {    color: black;}
.bracket {    color: blue;}
.css .property { color:22px;}
.css .numerical { color:seagreen;}
.css .letter { color:cornflowerblue;}
