Чекбоксы

Чекбоксы используются, когда у пользователя есть возможность выбрать любое количество вариантов (все, несколько или ничего).

Анатомия

Спецификация

Вёрстка

Чекбоксы могут быть расположены горизонтально или вертикально.

Рекомендуемое максимальное количество вариантов ответа для одной группы чекбоксов — не более 9. Одиночный чекбокс может быть использован для включения/выключения какого-то действия.

По умолчанию пользователь не видит выбранным ни один из вариантов ответа.

Содержание

Каждый чекбокс должен справа иметь лейбл, начинающийся с заглавной буквы. Использование в лейблах только заглавных и строчных букв не допускается. Длина лейбла — не более одной строки (45-70 символов), а желательно 2-3 слова;
