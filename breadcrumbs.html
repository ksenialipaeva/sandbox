Хлебные крошки

Хлебные крошки используются для визуализации положения пользователя в иерархии сайта, а также для быстрого выхода со страницы.

Спецификация

Максимальная длина названия страницы — 150px. В случает более длинного названия следует сокращать эллипсисом.

Если цепочка хлебных крошек не помещается в одну строчку, то её необходимо сократить с помощью добавления многоточия между первым и последним элементом.

<body>
    <pre><code class="language-html">...</code></pre>
</body>
.punctuation {    color: black;}
.bracket {    color: blue;}
.css .property { color:22px;}
.css .numerical { color:seagreen;}
.css .letter { color:cornflowerblue;}
